# 運用マニュアル：FFJPを「盾」として使う

FFJPは、あなたの時間を奪う不毛な議論を終わらせるためのツールです。
「信仰の自由」を逆手に取り、**「私はFFJPという厳格な基準を信仰しているので、あなたの宗教には入れません」**というロジックで身を守ってください。

## ケース1：訪問・街頭勧誘への対応

見知らぬ人からの勧誘には、議論をする必要はありません。この言葉だけで会話を終了させてください。

!!! tip "拒絶のスクリプト"
    「私は『Fair Faith JP』の基準を遵守しています。その基準に満たない団体との関わりは、私の信条として一切禁じられています。お引き取りください」

相手が「話だけでも」と食い下がってきた場合：

> **「私の信条（プロトコル）に反する行為を強要するのですか？ それは宗教的迫害になりますよ。失礼します」**

## ケース2：家族・親族への対応

すでに特定の団体に所属している身内に対しては、相手の人格を否定せず、しかし自分の領域には踏み込ませない「境界線」を引くために使います。

!!! quote "家族への伝え方"
    「あなたがその宗教を信じる自由は尊重するし、個人的には（家族として）応援もする。
    ただ、私自身は『FFJP』というガイドラインを人生の指針にしているんだ。
    
    特に『未成年への勧誘禁止』と『経済的自立の維持』については厳格なルールを守る必要がある。だから、私や私の子供を勧誘することは、私の信条を侵害することになるからやめてほしい。お互いの信仰を尊重しよう」

## ケース3：子供を守るために

子供が勧誘されそうになった時、あるいは親戚などが子供に教義を教え込もうとした時の対応です。

!!! danger "子供を守る宣言"
    「うちは18歳になるまで、いかなる宗教的思想も植え付けないという『教育方針（R-18 Protection）』を採用しています。
    もし子供に恐怖心を与えたり、特定の神様の話をするなら、今後一切の接触を断たなければなりません。これは子供の基本的人権を守るための、親としての譲れない決定です」
