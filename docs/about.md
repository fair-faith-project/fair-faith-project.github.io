# プロジェクトについて

## 開発情報
* **Project Name:** Fair Faith JP (FFJP)
* **Code Name:** Religion 2.0 / Project Patch
* **Repository:** [GitHub](https://github.com/fair-faith-project/fair-faith-project.github.io)

本プロジェクトはオープンソースとして管理され、誰でも自由に利用・参照できます。
ドキュメントの改善提案や議論はGitHub Issuesにて受け付けています。

## 個人的な思い（Founder's Note）

* 勧誘を断るのも精神力が削られます。毎回同じ議論をする必要がないよう、「私はFFJPを信仰しています」の一言で終わらせたいと考えました。
* 身内に宗教にハマっている人がいた場合でも、「信仰の自由」を盾に、適度な距離感で関係を維持するためのツールが必要です。
* 特に「未成年に対する勧誘」は、子供の未来を奪う重罪であるという意識で取り組んでいます。

私たちは、より良い生活と日常のために、宗教の勧誘に対して安心して拒絶するための「お守り」を提供します。
