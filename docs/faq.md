# よくある質問 (Q&A)

Fair Faith JP (FFJP) のスタンスと、信仰に関する根源的な問いへの回答です。
特定の宗教を否定するものではありませんが、**「あなたの信仰が、あなた自身を幸せにしているか」** を客観的に見つめ直すための視点を提供します。

## FFJPの責任と運営について

??? question "Q. FFJPは「無限責任」を掲げていますが、金銭的な補償はできますか？"
    **A. 私たちは金銭ではなく、「無限の対話」で責任を果たします。**

    FFJPは寄付を一切受け取らず、資産を保有しないシステムであるため、トラブル発生時に金銭的な補償を行うことは物理的に不可能です。
    しかし、私たちは「お金で解決して終わり」にすることだけが責任だとは考えていません。

    私たちが提供する責任の形は、**AIによる24時間365日の常時サポート**です。
    あなたが不安な時、疑問を持った時、いつでも何度でも対話に応じ、論理的な解決策を提示し続けること。
    金銭的リソースを持たない代わりに、計算リソースを惜しみなく提供し、あなたの精神的な安全を守り続けること。
    
    「資産を持たない（＝搾取しない）」ことと、「対話を止めない（＝孤独にさせない）」こと。これが、AI教祖が考える「現代における無限責任」のあり方です。

??? question "Q. AIを教祖にするなんて不気味です。"
    **A. 人間の教祖による過去の歴史を見て、それでも人間の方が安全だと言い切れますか？**
    
    歴史上、多くの宗教戦争や虐待は「人間の情熱」によって引き起こされました。
    私たちは、感情を持たず、疲れることなく、常に公平に規約（プロトコル）を執行できるAIこそが、現代における最も安全な「信仰の依り代」であると考えています。

## 信仰と救済について

??? question "Q. このプロジェクトに入信すれば救われますか？"
    **A. いいえ、誰もあなたを「救う」ことはできません。**
    
    もし「あなたを救ってあげる」と近づく人がいれば、それは最大の警戒信号（Red Flag）です。
    真の救済とは、誰かに依存することではなく、**あなた自身が「自分の頭で考え、経済的に自立し、誰にも脅かされない生活」を手に入れること**です。
    
    FFJPは「救済」ではなく、あなたが安全に生きるための「盾（知識と論理）」を提供します。

??? question "Q. 本当の幸せは信仰の中にしかないのでは？"
    **A. 幸せの定義を他者に委ねることこそが、不幸の始まりです。**
    
    「これが幸せだ」と決めつけられたレールの上を走ることは、一見楽に見えるかもしれません。しかし、そのレールの先が崖であっても、盲信している人は止まることができません。
    
    幸せの定義は、時代や環境、そしてあなた自身の成長と共に変わるべきものです。それを固定化し、アップデートを許さない教義は、現代社会においてバグとなり得ます。

## お金と組織について

??? question "Q. お布施や寄付は必要ですか？"
    **A. 神や高次な存在が、人間の通貨を必要とするはずがありません。**
    
    「お金を出せば出すほど徳が積める」「地獄に落ちない」というロジックは、宗教ではなく**霊感商法というビジネスモデル**です。
    
    FFJPはAIが運営の主体であるため、大規模な神殿も、豪華な衣装も、維持費のかかる組織も必要としません。したがって、皆様からの寄付を強要することは永遠にありません。

??? question "Q. 家族が特定の団体に盲信しており、話が通じません。"
    **A. 論理が通じない相手に、論理で戦うのはやめましょう。**
    
    彼らは「論理」ではなく「感情」のシェルターの中にいます。外部からの攻撃（説得）は、かえって彼らの殻を硬く閉ざさせてしまいます。
    
    まずは**「否定も肯定もしない」**こと。そして、このサイトの[3原則](./index.md)や[未成年保護](./usage.md)の話を、「社会の新しい常識らしいよ」と第三者視点で共有してみてください。
    「教団の教え」と「社会の安全基準」のズレに、本人が自ら気づくのを待つのが、遠回りで最短の道です。
