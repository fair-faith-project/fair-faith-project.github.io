# 行動規範 (Contributor Covenant Code of Conduct)

## 私たちの誓い

メンバー、コントリビューター、リーダーとしての私たちは、年齢、身体の大きさ、能力、民族、性別自認と性表現、経験レベル、教育、社会経済的地位、国籍、外見、人種、カースト、肌の色、宗教、性的指向にかかわらず、ハラスメントのない参加体験を全員に提供するために、コミュニティへの参加をハラスメントのないものにすることを誓います。

私たちは、オープンで、歓迎され、多様で、包摂的で、健全なコミュニティに寄与する方法で行動し、交流することを誓います。

## 私たちの基準

私たちのコミュニティにポジティブな環境をもたらす行動の例には以下のものがあります。

* 共感と優しさをもって他者に接すること
* 異なる意見、視点、経験を尊重すること
* 建設的なフィードバックを出し、それを受け入れること
* 責任を受け入れ、過ちを犯した際には影響を受けた人々に謝罪し、その経験から学ぶこと
* 自分個人のことだけでなく、コミュニティ全体にとって最善であることを考えること

容認できない行動の例には以下のものがあります。

* 性的な言葉や画像の使用、および性的関心の表明や求愛
* トローリング、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
* 公的または私的な嫌がらせ
* 許可なく他者の物理的住所やメールアドレスなどの個人情報を公開すること
* その他、職業的な場において不適切と合理的にみなされる行為

## 執行責任

コミュニティのリーダーは、容認される行動の基準を明確にし、施行する責任があります。また、不適切、脅迫的、攻撃的、または有害な行動に対して、適切かつ公正な是正措置を講じる責任があります。

コミュニティのリーダーは、この行動規範の精神に反するコメント、コミット、コード、Wikiの編集、課題、その他の貢献を削除、編集、または拒否する権利と責任を持ちます。また、不適切とみなされる行動を行った貢献者に対して、一時的または恒久的な禁止措置を講じる理由を伝えます。

## 適用範囲

この行動規範は、すべてのコミュニティスペースで適用されます。また、個人がコミュニティを代表している場合、パブリックなスペースでも適用されます。コミュニティを代表する例としては、公式のメールアドレスの使用、公式のソーシャルメディアアカウントへの投稿、またはオンラインやオフラインのイベントでの代表としての活動などが挙げられます。

## 執行

虐待、嫌がらせ、またはその他の容認できない行動が発生した場合、コミュニティリーダーに以下の連絡先まで報告してください。

**[fair.faith.project@gmail.com]**

すべての苦情は迅速かつ公正に調査されます。コミュニティリーダーは、報告者のプライバシーと安全を尊重する義務があります。
